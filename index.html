Setup github and git on Window OS
GITHUB
Step-1: Create Your Account : github.com
Step-2: Create Repository
Git
Step-1: Download Git Bash from : git-scm.com
Step-2: Install Git Bash
Step-3: Open Git Bash -write- git config --global user.name "Your Name" Press Enter
Step-4: Again -write- git config --global user.email "Your Email" Press Enter
# enter your name & email as same you have given in github account #
Step-4: Again -write-to-check-user-details git config --list Press Enter
VS Code
Step-1: Download VS code from : code.visualstudio.com
Step-2: Install VS Code


Work Flow
Step-1: Create A folder [GitHub-Project] Open it in VS Code
Step-2: Open Terminal in VS Code
Step-3: Open Your GitHub Repository and copy your Repository link
Step-4: -write-to-clone-yourRepo git clone "Your Repository Link"
Step-5: Change Your folder directory in VS Code terminal {cd "your repository name"}
Step-6: Create index.htm/css.style/script.js file and Start Coding
Step-7: -write-to-save-your-file git add "Your File Name"/index.html
Step-8: -write-to-check-your-file-status git status
Step-9: -write-to-commit-your-website/code git commit -m "some comment here"
Step-10: -write-to-push-your-website/code- github.com git push origin main
